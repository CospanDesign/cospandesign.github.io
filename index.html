Nysa

Nysa is an open-source platform that simplifies hardware and software development with FPGAs.

Speeds up FPGA-based project development.

Nysa's software tools reduce the unnecessary complexities when building and interfacing with FPGAs so that working with FPGAs does not hurt anymore. By providing a structured approach, it simplifies the deveopment workflow, opening up the powerful features of FPGAs to everyone.

Built-in infrastructure

Built-in standard interfaces for device communication so that you can focus on developing your new devices instead of wasting time on re-inventing the wheels.

Drag and drop UI

Create FPGA images from visual configuration using drag and drop, and then download them to a user FPGA. No knowledge of HDL required!

Simple API

Communicating with an FPGA using a simple Python API (read to register, write to register, etc...)
